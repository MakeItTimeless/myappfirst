# To-Do List Application (React.js)

A simple, efficient, and user-friendly To-Do List application built using **React.js**.  
This application allows users to manage daily tasks by adding, editing, deleting, and marking tasks as completed in real time.

---

## Overview

The To-Do List App is a front-end web application developed using React.js.  
It demonstrates the use of React functional components, state management using hooks, and dynamic rendering of UI elements.

---

## Features

- Add new tasks  
- Edit existing tasks  
- Delete tasks  
- Mark tasks as completed  
- Real-time UI updates  
- Responsive design  

---

## Tech Stack

- **Frontend:** React.js  
- **Language:** JavaScript (ES6+)  
- **Styling:** CSS  
- **Package Manager:** npm

## Usage

To start the application, run:
npm start

The application will run on:
http://localhost:3000


---

## Application Workflow

1. User enters a task in the input field  
2. Task is stored in the component state using `useState`  
3. Tasks are displayed dynamically using `.map()`  
4. User can:
   - Edit a task
   - Delete a task
   - Mark a task as completed  
5. UI updates instantly without page refresh
